# Windows LAB Environment Overview

This documentation describes a lab-scale Windows infrastructure deployed on Hyper-V. It is designed to demonstrate network segmentation, firewall management, and Active Directory domain services in a professional lab environment.

## Objectives

- Build a Hyper-V lab with separate WAN and production network segments.
- Deploy OPNsense as the perimeter firewall and gateway.
- Deploy a Windows Server Domain Controller with Active Directory, DNS, and DHCP.

## Lab scope

- Hyper-V virtual switch configuration
- OPNsense firewall installation and LAN/WAN setup
- Windows Server domain controller installation, promotion, and service configuration

- [Installing OPNsense](Installing-OPNsense.md)

## Summary

This lab is intended to demonstrate foundational Windows infrastructure skills, including network segmentation, secure firewall placement, and centralized identity and service management.
